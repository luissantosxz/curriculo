Apostila de HTML

Html:

Html é uma linguagem de marcação usada principalmente para web sites

CSS:

CSS é usado para "estilizar" a HTML, assim podendo usar CORES(RGB),e também separar a representação visual do site, podendo escolher o alinhamento de componentes no site,etc

Tags HTML:

  • Aqui são alguns exemplos de tags HTML:
  • <p> É uma tag para parágrafos
  • <h1> É uma tag para Titulos
  • <h2> É uma tag para subtitulos
  • <strong> É uma tag para deixar em negrito
  • <div> Serve para "encapsular" outras tags
  • <section> É uma tag que define uma "sessao"

Exemplo de uma estrutura de HTML: 

        
        <!DOCTYPE html> 
        <html> 
        <head> 
        <meta charset="UTF-8"/> 
        <title>Document</title> 
        </head> 
        <body> 
        <!-- Conteúdo --> 
        </body> 
        </html>
