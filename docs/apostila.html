Sobre HTML

O HTML é uma linguagem de marcação essencialmente utilizada na criação de sites, permitindo a estruturação e formatação dos conteúdos online. É a base que torna a web visualmente atrativa e funcional.

Sobre CSS

O CSS desempenha um papel crucial na web ao permitir a estilização da HTML, possibilitando o uso de cores, ajustes de layout e separando a apresentação visual dos elementos, o que torna a criação de sites mais flexível e personalizada. Ele oferece controle detalhado sobre a aparência e o posicionamento dos elementos em uma página web.

Tags HTML

Aqui estão alguns exemplos de tags HTML:

  • <p> É uma tag para parágrafos
  • <h1> É uma tag para Titulos
  • <h2> É uma tag para subtitulos
  • <strong> É uma tag para deixar em negrito
  • <div> Serve para "encapsular" outras tags
  • <section> É uma tag que define uma "sessao"

Exemplo de uma estrutura de HTML

Aqui está o exemplo da estrutura:

                    
<!DOCTYPE html> 
<html> 
<head> 
<meta charset="UTF-8"/> 
<title>Document</title> 
</head> 
<body> 
<!-- Conteúdo --> 
</body> 
</html>

Aqui está o exemplo de código HTML:

                    
<h1> Olá, mundo</h1> 
<p> Este é um paragráfo.</p>
